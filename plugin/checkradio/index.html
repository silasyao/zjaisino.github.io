    组件 - 模拟单选复选框

    使用说明
    							
							<link rel="stylesheet" type="text/css" href="checkradio.css" />
							
                            
                            <script type="text/javascript" src="jquery.js"></script>
                            <script type="text/javascript" src="jquery.checkradio.js"></script>
							
							
							<script>
							$(':checkbox, :radio').checkradio(参数);
							</script>
    发送验证码

    单选框

    男 女 
                                <span class="ui-checkradio">
								<input type="radio" name="sex" value="1" />男
							</span>
							<span class="ui-checkradio">
								<input type="radio" name="sex" value="2" />女
							</span>
                            
                                $(':radio').checkradio();

    复选框

    全选
    html css js 
                                <span class="ui-checkradio">
								<input type="checkbox" name="all" /> 全选
							</span>
							<div>
								<span class="ui-checkradio">
									<input type="checkbox" name="lang[a]" /> html
								</span>
								<span class="ui-checkradio">
									<input type="checkbox" name="lang[a]" checked /> css
								</span>
								<span class="ui-checkradio">
									<input type="checkbox" name="lang[a]" /> js
								</span>
							</div>
                            
                                $('[name="all"]').checkradio({
								callback:function(ele, e){
									$('[name="lang"]').checkradio('checked', ele.prop('checked'))
								}
							});
							
							$('[name="lang"]').checkradio({
								callback:function(ele, e){
									$('[name="all"]').checkradio('checked', $('[name="lang"]').length === $('[name="lang"]:checked').length);
								}
							});

    开关按钮

    							<span class="ui-checkradio">
								<input type="checkbox" name="switch1" data-off="OFF" data-on="ON" /> 
							</span>
							
                            <span class="ui-checkradio">
								<input type="checkbox" name="switch2" checked />
							</span>
                            
    							$('[name="switch1"]').checkradio();
							
                            $('[name="switch2"]').checkradio({
								switches:{
									off:'未开启',
									on:'已开启'
								}
							});
    API
    属性名称 值类型 默认值 可填值 功能说明
    switches Object { off : ' ', on : ' ' } { off : 值, on : 值 } 配置开关按钮，off和on不能为空，也可在元素上设置data-off data-on属性，设置后将会覆盖原值
    方法名称 参数 默认值 返回值 功能说明
    beforeCallback ele 调用组件jq对象 undefined | false 点击元素状态改变前回调，如返回值为false则将阻止状态改变
    event 事件对象
    callback ele 调用组件jq对象 undefined 点击元素状态改变后回调
    event 事件对象
    备注
    $(selector).checkradio('checked', true|false) 或者 $(selector).checkradio('disabled', true|false)可改变元素状态

    组件 - 限制输入

    使用说明
                                
                            <script type="text/javascript" src="jquery.js"></script>
                            <script type="text/javascript" src="jquery.numeral.js"></script>
    限制5位整数
                                $('.number').numeral({
                                integer:5,
                                decimal:false
                            });
    限制3位整数，保留2位小数，可以正负切换

    10 * = 

                                $('.decimal').numeral({
                                integer:3,
                                minus:true,
                                convert:true,
                                callback:function(elem, val){
                                    var value = '';
                                    if(val){
                                        value = (10 * val).toFixed(2);
                                    }
                                    elem.siblings('b').text(value);
                                }
                            });
    API
    属性名称 值类型 默认值 可填值 功能说明
    integer Number 4 正整数 整数长度，值为0则文本框可以输入任意字符
    decimal Number 2 正整数 小数长度，值为0则不启用小数
    minus Boolean false true | false 是否可以输入负数
    convert Boolean false true | false 是否启用正负切换，值为负数，输入“ - + ”转换为正数
    方法名称 参数 默认值 功能说明
    callback elem 调用组件jq对象 键盘抬起时的回调
    value 输入框的值

    使用说明

                            <script type="text/javascript" src="jquery.js"></script>
                            <script type="text/javascript" src="jquery.superslide.js"></script>
                            
                            
                            $('.demo').superSlide(配置参数);

    容器建议加上position:relative样式，因为内部定位元素是动态创建的，在网速慢时，js还没加载完成之前，布局会乱

    子元素必须包含类名ui-slide

    加载图片可用data-src属性懒加载

    实例

                                    isHide:true

    API
    普通属性 值类型 默认值 可填值 功能说明
    delay Number 4000 正整数 自动切换延迟时间，1000=1s
    speed Number 450 正整数 切换速度
    event String click click/mouseover 圆点按钮触发事件
    ease String null easing动画类型 缓动效果，依赖jquery.easing.js
    itemNum Number 0 正整数 一屏显示数量，<ul class="ui-slide"></ul>中不要使用li标签，会自动生成
    moveTo Number 0 正整数 默认展示第几个
    isBackdrop Boolean false true|false 切换图片为背景图片，<li data-src="背景图片路径"></li>
    isFadein Boolean false true|false 是否以淡入淡出效果显示
    isAuto Boolean true true|false 是否自动切换
    isHorz Boolean true true|false 是否水平切换
    isTitle Boolean false true|false 是否显示标题，<li data-title="文本或者html"></li>
    对象属性 属性名 值类型 默认值 可填值 功能说明
    list enable Boolean true true|false 是否展示圆点按钮
    isHide Boolean false true|false 是否默认隐藏，鼠标悬停后显示
    button enable Boolean true true|false 是否展示左右切换按钮
    isHide Boolean false true|false 是否默认隐藏，鼠标悬停后显示
    thumb enable Boolean false true|false 是否展示相册底部缩略图
    isHorz Boolean true true|false 缩略图是否水平切换
    resize enable Boolean false true|false 是否自适应切换
    target Object $(window) jQuery Object 自适应目标元素
    minSize Number 0 正整数 水平切换表示最小宽度，垂直切换表示最小高度
    callback Function 空函数 function(){...} 窗口大小改变后回调函数
    scroll enable Boolean false true|false 是否开启非循环滚动
    isEnd Boolean false true|false 是否默认显示最后一屏
    isEvent Boolean false true|false 列表中元素是否触发事件，配合callback函数使用
    普通方法 参数 参数说明 返回值 功能说明
    callback index 当前展示item索引 - 切换过渡回调函数，若为非循环滚动，表示li标签触发event后的回调函数
    item 当前展示item -
    dot 当前展示选中点控制器 -
    endCallback index 目标元素索引 - 切换完成回调函数
    item 当前展示item -
    dot 当前展示选中点控制器 -

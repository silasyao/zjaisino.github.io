    组件 - 输入框占位符

    使用说明
                                
                            <input type="text" name="demo" placeholder="占位符内容" />
                            
                            
                            <script type="text/javascript" src="jquery.js"></script>
                            <script type="text/javascript" src="jquery.placeholder.js"></script>
                            
                            
                            $('[name="demo"]').placeholder(配置参数);
    该组件会增加输入框的父元素strong标签，以及兄弟元素b标签
    实例

    动画版

                                <div class="m-demo">
                                <input type="text" name="demo1" class="ui-input" placeholder="请输入用户名" />
                            </div>
                            
                                $('[name="demo1"]').placeholder({equal:true});

    传统版

                                <div class="m-demo">
                                <input type="password" name="demo2" class="ui-input" placeholder="请输入密码" />
                            </div>
                            
                                $('[name="demo2"]').placeholder({animate:false});
    API
    普通属性 值类型 默认值 可填值 功能说明
    animate Boolean true true/false 是否启用动画版
    equal Boolean false true/false 输入框值是否可以和占位符相同，如果为false，当两者相同时，值会清空
    color String #ccc 十六进制颜色值 占位符文本颜色
    theme String '' 英文字符串 类选择器值，可对占位符设置样式

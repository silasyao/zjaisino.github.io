    组件 - 下拉菜单

    使用说明
                                
                            <select name="demo" data-init="函数名" data-func="函数名" data-value="默认值" data-css="样式"></select>
                            
                            
                            <script type="text/javascript" src="jquery.js"></script>
                            <script type="text/javascript" src="jquery.select.js"></script>
                            
                            
                            $('[name="demo"]').imitSelect(配置参数);
    select元素上的data属性为可选项
    若使用ajax加载数据，请设置为同步请求
    data-init：当组件第一次点击后触发
    data-func：组件每次选择后触发
    data-value：设置默认选中值，可配置参数中value属性功能相同
    data-css：定义组件样式，通常为设置宽度
    实例

    多级联动

                                <select class="demo3" name="prov" data-css="width:100px" data-init="getProv" data-value="1">
                                <option value="0">请选择</option>
                            </select>
                            <select class="demo3" name="city" data-css="width:100px" data-func="getCity" disabled="disabled">
                                <option value="0">请选择</option>
                            </select>
                            <select class="demo3" name="area" data-css="width:100px" data-func="getArea" disabled="disabled">
                                <option value="0">请选择</option>
                            </select>
                            
                                function getProv(value){
                                if(value !== undefined){
                                    if(value != '0'){
                                        $('select[name="city"]').imitSelect({isInit:false, disabled:false, value:value});
                                    }
                                    else{
                                        $('select[name="city"]').imitSelect({isInit:false, disabled:true, isNull:true});
                                    }
                                    $('select[name="area"]').imitSelect({isInit:false, disabled:true, isNull:true});
                                }
                                else{
                                    var option = '<option value="0">请选择</option>';
                                    for(var k in GP){
                                        option += '<option value="'+ (parseInt(k)+1) +'">'+ GP[k] +'</option>';
                                    }
                                    return option;
                                }
                            }
                            
                            function getCity(value){
                                if(value !== undefined){
                                    if(value != '0'){
                                        $('select[name="area"]').imitSelect({isInit:false, disabled:false, value:value});
                                    }
                                    else{
                                        $('select[name="area"]').imitSelect({isInit:false, disabled:true, isNull:true});
                                    }
                                }
                                else{
                                    var option = '<option value="0">请选择</option>';
                                    var data = GT[$('select[name="prov"]').val()-1];
                                    for(var k in data){
                                        option += '<option value="'+ (parseInt(k)+1) +'">'+ data[k] +'</option>';
                                    }
                                    return option;
                                }
                            }
                            
                            function getArea(value){
                                if(value === undefined){
                                    var option = '<option value="0">请选择</option>';
                                    var data = GC[$('select[name="prov"]').val()-1][$('select[name="city"]').val()-1];
                                    for(var k in data){
                                        option += '<option value="'+ (parseInt(k)+1) +'">'+ data[k] +'</option>';
                                    }
                                    return option;
                                }
                            }

    API
    普通属性 值类型 默认值 可填值 功能说明
    count Number 8 任意正整数 下拉显示个数
    animate String '' css类选择器值 添加css3动画类
    selext String '' html字符串 添加自定义结构
    value String/Number/Null null - 设置下拉框默认值
    disabled Boolean false false/true 是否禁用下拉框
    isInit String/Number/Null null - 是否初始化默认值，配合value属性使用
    isNull Boolean false false/true 将value值清空
    isEdit Boolean false false/true 下拉框是否可编辑
    hide Boolean false false/true 是否隐藏下拉框
    isTitle Boolean false false/true 下拉框列表是否有title提示
    对象属性 属性名 值类型 默认值 可填值 功能说明
    search enable Boolean false false/true 是否开启搜索功能
    tips String '' 任意字符串 搜索默认占位符文本，配合jqyery.placeholder组件使用
    text String '' 任意字符串 未搜索到结果提示文本
    普通方法 参数 参数说明 返回值 功能说明
    callback target 调用组件的select JQ对象 - 点击列表回调函数，初始化时会调用一次
    filter text 下拉框选中文本 - 过滤文本函数

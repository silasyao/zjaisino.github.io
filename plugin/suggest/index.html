    组件 - 搜索提示

    使用说明
                                
                            <link rel="stylesheet" type="text/css" href="suggest.css" />
                            
                            
                            <input type="text" name="search" />
                            
                            
                            <script type="text/javascript" src="jquery.js"></script>
                            <script type="text/javascript" src="jquery.suggest.js"></script>
                            
                            
                            $('[name="search"]').suggest(配置参数);

    接口返回结果格式必须为['结果1', '结果2', ...] 或者 [{结果1}, {结果2}, ...]

    支持键盘方向键上下操作

    实例
                                <input type="text" name="search" class="u-input" />
                            
                                $('[name="search"]').suggest({
                                url:'http://www.yinjiazeng.com/testdata/test/suggest/index.php?callback=?',
                                returnData:function(item){
                                    return '
  • '+ item.name +'
    • ';
                                }
                            });
    API
    普通属性 值类型 默认值 可填值 功能说明
    url String '' url地址 ajax请求url
    param String keywords 英文字符串 url接口接收参数
    theme String '' 英文字符串 自定义组件主题
    getDom String '' 英文字符串 指定下拉列表文本，将列表指定元素内容赋值到输入框中
    limit Number 10 正整数 下拉数据展示数量，多出则滚动显示
    select Boolean false true/false 是否默认选中第一行列表
    cache Boolean true true/false 将对应关键词查询到的数据保存到js中，下次再搜索该词将不调用接口
    普通方法 参数 参数说明 返回值 功能说明
    getData keywords 关键字 列表数据 若没有设置url参数，将启用该方法
    returnData item 单行源数据 处理后的列表li元素 单行源数据
    eventEnd item 触发事件行jQuery对象 - 选择列表行触发函数
    target 目标输入框jQuery对象
    callback target 目标输入框jQuery对象 - 初始化组件完成回调函数
    suggest 下拉框jQuery对象
    hideEvent target 目标输入框jQuery对象 - 隐藏下拉框回调函数
    suggest 下拉框jQuery对象

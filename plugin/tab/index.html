        API
        属性名 值类型 默认值 可填值 功能说明
        event String click click/mouseover 触发事件
        auto Boolean false true/false 是否自动切换
        delay Number 4000 正整数 自动切换延迟时间，1000=1秒
        index Number 0 正整数 默认展示菜单索引
        content String '' 字符串 指定ajax数据容器元素，为空则指向最外层容器
        方法名 参数 参数说明 返回值 功能说明
        ajax index 展示状态选项卡索引 Undefined ajax请求数据
        item 展示内容区域容器对象
        callback index 展示状态选项卡索引 true/false 展示内容之前回调函数，返回false则不展示对应内容区域
        item 展示内容区域容器对象
        endCallback index 展示状态选项卡索引 Undefined 展示内容之后回调函数
        item 展示内容区域容器对象
        DOM对象方法名 参数 参数说明 返回值 功能说明
        addItem data [{tabmenu:'', tabcon:''}, ...] 添加的最后一个选项卡菜单 添加选项卡
        removeItem index 移除菜单索引 Undefined 删除选项卡
        nextall 是否移除指定选项卡之后所有菜单
